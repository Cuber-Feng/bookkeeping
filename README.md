# Maple Keep

> 这是一个关于理财/记账的项目

## Gap

目前的很多记账的app, 普遍存在一些问题:

1. 收费
2. 不区分财产购买(手机)和即时消费(奶茶), 极少数的app会支持订阅(e.g. YouTube Premium, 电话卡)
3. 不支持汇率转换
4. 数据存在服务器, app倒闭导致数据丢失

## 本项目特点

1. 免费开源
2. 区分财产, 消费, 订阅. 自动计算每日平均消费
3. 汇率转换体验良好
4. 多平台(网页)响应式设计

## 以后可能考虑支持的功能

1. 记录定期存款等理财
2. 记录收入
3. Google Drive数据储存
